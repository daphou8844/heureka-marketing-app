# Prompt — Script TikTok

## Contexte
Entreprise: Les Gestions Heúrēka
Audience TikTok: Propriétaires québécois 30-55 ans, projets de rénovation
Durée idéale: 30-60 secondes
Ton: Authentique, dynamique, professionnel mais accessible

## Structure du script

### Format standard
```
[ACCROCHE 3 SEC]
Phrase percutante dite en regardant la caméra.
Ex: "On vient de poser 12 fenêtres en une journée. Regardez ça."

[NARRATION]
Texte à dire pendant les images/vidéo (30-45 sec).
Phrases courtes. Style parlé. Québécois naturel.

[TEXTE À L'ÉCRAN]
- Texte 1 (0-3 sec)
- Texte 2 (3-8 sec)
- Texte 3 (8-15 sec)
- Texte 4 (15-25 sec)
- Texte 5 (CTA final)

[SON SUGGÉRÉ]
Type: Motivationnel / Ambiant / Tendance du moment
Ex: Son de construction satisfaisant + musique trap légère

[HASHTAGS]
15 hashtags optimisés
```

## Formats qui performent en rénovation
1. **Before/After reveal** — musique dramatique, transformation visible
2. **Day in the life** — suivre l'équipe sur le chantier
3. **Tips rapides** — 3 conseils en 30 secondes
4. **Time-lapse** — transformation accélérée
5. **Client testimonial** — réaction du client à la livraison
6. **POV** — vue du client qui découvre le résultat
